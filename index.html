Instructions
Alea's Snake est un jeu solo. Vous devez contrôler le serpent dans le cadre bordé.
Il vous suffit d'utiliser les touches fléchées du clavier pour déplacer le serpent dans différentes directions. Le jeu continue jusqu'à ce que vous sauviez le serpent du serpent lui-même.
